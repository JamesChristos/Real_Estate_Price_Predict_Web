# UrbanRealty - README

Welcome to the UrbanRealty repository. This project aims to help customers find their dream houses by providing an easy-to-use interface for estimating home prices based on specific parameters.

## Features

- **Home Price Prediction**: Enter details such as total square feet, location, number of bathrooms, and number of bedrooms to get an estimated price.
- **Wide Coverage**: The platform covers over 242 locations with more than 7,251 properties listed.
- **User-Friendly Interface**: Simple and intuitive UI for users to easily input their requirements and get quick results.
- **Statistics Display**: Displays various statistics, including the number of locations and properties sold.

## Installation

To run this project locally, follow these steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/UrbanRealty.git

2. **Navigate to the project directory**
   ```bash
   cd UrbanRealty

3. **Install the necessary dependencies**
   ```bash
   npm install

## Usage

1. **Start the development server**
   ```bash
   npm start

2. **Open your web browser and go to**
   ```bash
   http://localhost:3000

## Components
### Home Price Prediction Modal
- **Total Square Feet:** Input field to enter the total square feet of the property.
- **Location:** Dropdown menu to select the location (e.g., 1st block Jayanagar).
- **Bathrooms:** Input field to enter the number of bathrooms.
- **Bedrooms:** Input field to enter the number of bedrooms.
- **Estimated Price:** Displays the estimated price based on the entered parameters.
- **Predict Button:** Triggers the prediction process.
- **Cancel Button:** Closes the prediction modal without making any changes.
